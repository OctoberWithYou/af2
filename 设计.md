## 模型调用代理转发项目

### 项目概述

AI Forward 是一个用于在内网部署 Agent，通过公网 Server 转发 AI 模型访问请求的系统。

### 技术栈

| 模块 | 技术栈 |
|------|--------|
| common | Java 17, Jackson |
| server | Java 17, Spring Boot 3.5, Netty 4.1 |
| agent | Java 17, Netty 4.1 |
| deploy | Java 17, Spring Boot 3.5, MyBatis, H2, **Vue 3.5 + TypeScript 5.7 + Vite 6** |
| test | JUnit 5, RestAssured, **Vitest**, @vue/test-utils |

### 快速启动

```bash
# 构建项目
./gradlew build

# 运行测试
./gradlew test

# 生成代码覆盖率报告
./gradlew :test:jacocoTestReport

# 打包分发版本
./gradlew packageDistribution

# 启动 Server
./gradlew :server:bootRun

# 启动 Agent
./gradlew :agent:bootRun

# 启动 Deploy（含前端页面 http://localhost:8081/）
./gradlew :deploy:bootRun

# 前端开发模式
cd deploy/src/frontend
pnpm run dev
```

### 分发内容

执行 `./gradlew packageDistribution` 后，构建产物位于 `build/distribution/`：

```
build/distribution/
├── agent/           # Agent 模块 JAR 和配置
├── server/          # Server 模块 JAR 和配置
├── deploy/          # Deploy 模块 JAR 和配置
└── bin/             # 启动脚本
    ├── start-agent.sh
    ├── start-server.sh
    └── start-deploy.sh
```

### 前端技术栈

Deploy 模块前端采用现代技术栈：

- **框架**: Vue 3.5 + TypeScript 5.7
- **构建工具**: Vite 6
- **UI 组件库**: Element Plus 2.9
- **路由**: Vue Router 4.5
- **HTTP 客户端**: Axios
- **测试框架**: Vitest + @vue/test-utils
- **代码规范**: ESLint 9 + Prettier 3
