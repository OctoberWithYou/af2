## 测试覆盖率目标

- 单元测试覆盖率：>= 90%
- API 测试覆盖率：>= 90%
- 所有测试用例必须通过才能提交

## 测试运行命令

```bash
# 运行所有测试
./gradlew test

# 运行特定模块测试
./gradlew :common:test
./gradlew :test:test

# 运行特定测试类
./gradlew :test:test --tests "org.ljc.test.common.CommonModelTest"

# 生成 Allure 报告
allure serve build/reports/allure-results
```
